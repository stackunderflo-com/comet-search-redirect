This directory contains HTML mockups for creating professional Chrome Web Store screenshots.

## Quick Start

1. **Open files in Chrome:**
   ```bash
   start chrome "screenshot-1-hero.html"
   start chrome "screenshot-2-popup.html"
   start chrome "screenshot-3-examples.html"
   start chrome "screenshot-4-debug.html"
   start chrome "screenshot-5-comparison.html"
   ```

2. **Set browser to 1280x800px**
3. **Press F11 for fullscreen**
4. **Take screenshot (Win+Shift+S)**
5. **Save as PNG**

## Chrome Web Store Requirements

- **Size:** 1280x800px (recommended) or 640x400px
- **Format:** PNG or JPG
- **Max screenshots:** 5
- **Purpose:** Showcase key features and benefits
